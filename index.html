Proyek

Website CCTV11

Website CCTV11 merupakan proyek pengembangan website yang dibuat dalam rangka pembelajaran berbasis proyek (Project-Based Learning) dengan tujuan membangun sebuah sistem informasi pemasaran dan promosi produk CCTV secara digital. Website ini dirancang untuk memberikan informasi yang lengkap, jelas, dan mudah dipahami mengenai berbagai jenis produk CCTV, fitur keamanan yang ditawarkan, serta layanan pemasangan dan dukungan teknis. Pengembangan website dilakukan menggunakan teknologi HTML, CSS, dan JavaScript dengan fokus pada tampilan antarmuka yang modern, responsif, dan user-friendly. Desain visual dibuat agar nyaman diakses melalui berbagai perangkat, baik desktop maupun mobile, sehingga dapat meningkatkan pengalaman pengguna. Melalui proyek ini, saya menerapkan konsep dasar pengembangan web, mulai dari perancangan struktur halaman, pengelolaan konten, hingga pengaturan tampilan visual. Proyek Website CCTV11 tidak hanya bertujuan sebagai media promosi, tetapi juga sebagai sarana penerapan kemampuan teknis dalam konteks dunia nyata, khususnya pada bidang sistem keamanan berbasis teknologi informasi.

Sistem Administrasi Bengkel

Analisis dan Perancangan Sistem Informasi Administrasi Bengkel merupakan proyek yang bertujuan untuk meningkatkan efisiensi pengelolaan operasional bengkel melalui penerapan sistem informasi terkomputerisasi. Proyek ini dilatarbelakangi oleh masih digunakannya sistem pencatatan manual yang berpotensi menimbulkan keterlambatan, kesalahan pencatatan data, serta kesulitan dalam penyusunan laporan. Tahapan proyek meliputi analisis kebutuhan sistem, identifikasi permasalahan, serta perancangan solusi menggunakan berbagai diagram pemodelan sistem seperti Use Case Diagram, Activity Diagram, Class Diagram, Entity Relationship Diagram (ERD), dan Sequence Diagram. Perancangan basis data dilakukan menggunakan SQL Server 2019 untuk memastikan integritas dan keakuratan data. Hasil dari proyek ini diharapkan mampu membantu petugas administrasi dan teknisi bengkel dalam mengelola data pelanggan, kendaraan, layanan servis, suku cadang, serta transaksi pembayaran secara terintegrasi. Proyek ini juga menjadi sarana penerapan kemampuan analisis sistem dan perancangan basis data sebagai bekal dalam menghadapi kebutuhan dunia industri.
